# 💼 Anand-CV: Professional Portfolio Template

A modern, highly responsive, and interactive curriculum vitae (CV) and personal portfolio website. Designed to showcase professional experience, educational background, programming skills, and personal interests in a visually appealing and organized layout.

---

## 🚀 Features

*   **📱 Fully Responsive Design:** Optimized for mobile, tablet, and desktop viewports.
*   **🎨 Clean & Premium Aesthetics:** Structured layouts with modern typography and sleek color transitions.
*   **📊 Skill Visualization:** Interactive progress bars showcasing proficiency levels.
*   **⏳ Interactive Education Timeline:** Highlights academic history with smooth animations.
*   **🌟 Interactive Hover States:** Modern hover effects that dynamically fade out non-focused sections to highlight your current selection.

---

## 🛠️ Tech Stack

*   **HTML5:** Structured semantic markup.
*   **CSS3:** Vanilla CSS featuring custom properties (CSS variables), CSS Flexbox, CSS Grid, and custom animations.
*   **FontAwesome:** Modern icons for professional representation.

---

## 📁 Repository Structure

*   [`index.html`](file:///d:/github/Anand-CV/index.html): The core structure containing CV content.
*   [`style.css`](file:///d:/github/Anand-CV/style.css): Custom stylesheet defining colors, typography, transitions, and media queries.

## ⚙️ Configuration & Customization

To personalize this template for your own CV/portfolio:

1.  **Modify Details:** Open [`index.html`](file:///d:/github/Anand-CV/index.html) and replace the placeholder text under `Contact`, `Education`, `Programming skills`, `Experience/Projects`, and `Personal Information` with your own information.
2.  **Update Profile Picture:** Place your WebP or PNG format picture in the `Img/` directory and update the `src` attribute in line 16 of `index.html`.
3.  **Adjust Styling:** Change the core variables (such as colors and theme values) in [`style.css`](file:///d:/github/Anand-CV/style.css) inside the `:root` selector:
    ```css
    :root {
        --profile-theme: #4682bf;
        --timeline-circle-theme: #14253e;
        --color-tiffany: #00a6a6;
    }
    ```
